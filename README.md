## Redux Toolkit Course

```sh
$ git clone https://github.com/webmasterdevlin/redux-toolkit-course.git
$ cd redux-toolkit-course
$ npm install
$ npm run start:fullstack
```

The React app, and the fake web service will run concurrently.
